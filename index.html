Built a Python-based wireless auditing tool that automates network discovery, monitor mode configuration, and captures handshakes for WPA/WPA2 analysis. Integrated with Hashcat for offline password strength testing against captured hashes.
PythonWireless SecurityWPA2HashcatOffensive Security
