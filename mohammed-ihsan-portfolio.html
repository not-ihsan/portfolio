I am a 4th-semester Computer Security Investigations (CSI) student at Fleming College, passionate about understanding how attackers operate and building defences to stop them. My focus is on network security, vulnerability assessment, and ethical hacking in real-world environments.

I am equally comfortable in Linux and Windows environments.

I am actively seeking an internship where I can contribute to security operations, vulnerability assessments, or ethical hacking engagements while continuing to grow my skills in a professional setting.

Network Security

Intrusion Detection System Using SNORT

Installed and configured SNORT as a Network Intrusion Detection System (NIDS), wrote custom detection rules, monitored real-time traffic, and detected attacks including port scanning and ICMP flood attempts.

  • Configured SNORT NIDS and integrated with live network interface
  • Wrote and tested custom rules to detect suspicious activity patterns
  • Analysed alert logs and documented attack signatures
  • Recommended basic mitigation techniques for identified threats
Reflection
This project gave me practical experience in the full NIDS workflow — from deployment and rule writing to alert analysis and reporting. Learning to tune detection rules to minimize false positives while catching real threats is a foundational skill for any SOC or security analyst role.

NASA APOD Viewer GUI

Developed a Python-based graphical application using NASA's public REST API to fetch and display the Astronomy Picture of the Day, featuring date navigation, HD image download, and automatic caching.

  • Integrated NASA's public REST API for live data retrieval
  • Built a Tkinter GUI with previous/next date navigation

  • Added HD image download functionality for offline use

Wi-Fi Security

Wi-Fi Auditing Script

Comprehensive wireless network auditing tool featuring AP/station discovery, WPA2 handshake capture, and Layer 1 & 2 DoS simulation including Authentication flood, Deauthentication, and Beacon flood attacks.

  • Scanned nearby Wi-Fi networks: BSSIDs, SSIDs, channels, connected stations
  • Captured WPA2 handshakes for offline credential assessment
  • Simulated Layer 2 DoS: Authentication flood, Deauthentication, Beacon flood
  • Implemented Layer 1 DoS with user-selectable channel or full frequency band
Reflection
This project required deep understanding of the 802.11 protocol stack and the mechanisms behind wireless attacks. It reinforced the importance of responsible disclosure and ethical boundaries in security testing — skills central to professional penetration testing work.
